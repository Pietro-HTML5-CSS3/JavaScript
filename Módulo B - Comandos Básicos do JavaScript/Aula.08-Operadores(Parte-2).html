Operadores Relacionais


(>) -> Maior

(<) -> Menor

(>=) -> Maior ou igual

(<=) -> Menor ou igual

(==) -> Igualdade

(!=) -> Diferente

5 > 2 -> True

7 < 4 -> False

8 >= 8 -> True

9 <= 7 -> False

5 == 5 -> True

4 != 4 -> False

Operadores Identidade


(==) -> Igualdade

(===) -> Igualdade Restrita

5 == 5 -> True

5 == '5' -> True

5 === '5' -> False

5 === 5 -> True

Operadores Lógicos


(!) -> Diferente

(&&) -> Conjunção

(||) -> Disjunção

(&&) -> Tem que ser tudo verdadeiro

(||) -> Apenes um precisa ser verdadeiro

Precedência

Operadores Áritimeticos -> | () | ** | * | / | % | + | - | -> Operadores Relacionais -> O que aparecer primeiro -> Operadores Lógicos -> ! -> && -> ||

Operador Ternário

Teste -> (?) -> True -> (:) -> False
